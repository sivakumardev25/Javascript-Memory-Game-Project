# Memory Game

A simple and responsive Memory Matching Card Game built using **HTML**, **CSS**, and **JavaScript**.

# How to Play

- Click any card to flip it.
- Flip another card to find a matching pair.
- If both cards match, they stay flipped.
- If not, they flip back after a short delay.
- Continue until all pairs are found!
- Use the **Restart** button anytime to start a new game.

# Features

- DOM-based card flipping
- Random shuffle every game
- Match detection logic
- Responsive grid layout
- Clean and easy-to-understand code

# How to Run

1. Download or clone the project folder.
2. Open the file **index.html** in your browser.
3. Start playing!

# Project Files

- `index.html` — structure
- `style.css` — design and layout
- `main.js` — game logic
